# Getting Started

Ensure you have Python 3.6 with pip installed.

```
$ pip3 install virtualenv
$ virtualenv dev.venv
$ source dev.venv/bin/activate # on POSIX systems
$ .\dev.venv\Scripts\activate.ps1 # on Windows
$ pip3 install -r requirements.dev.txt
```

## Run tests

```
$ tox
```
